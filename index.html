一、课程简介

Integrated Math Pathway 第三阶段

Math III 学什么？

Integrated Math III 是 Integrated Math 路径中的第三阶段，通常延续 Math I 与 Math II 的 algebra、geometry、functions、statistics 与 modeling。它不只是“Algebra II 换名字”，而是把函数、方程、图像、三角、数据分析和真实情境建模放在同一条学习线上。

在南加州与加州公立高中中，不同学区可能把一学年拆成 Math IIIA 和 Math IIIB。常见安排是：IIIA 更集中在 function families 与代数结构；IIIB 更集中在 trigonometry、unit circle、statistics、normal distribution 与 introductory inference。

为什么重要？

Math III 的目标是让学生能熟练分析函数特征、解释参数意义、建立模型、解方程，并能用统计语言描述不确定性。它直接连接后续课程：Precalculus 需要扎实的 functions 与 trigonometry；AP Statistics 需要数据、模拟和推断思维；AP Calculus 则要求学生对图像、变化率和函数行为有稳定直觉。

本页参考 California Mathematics Framework 中 Mathematics III 的课程方向，以及 IUSD 对 Integrated Math Pathway 的说明：综合路径会在每一年整合 algebra、geometry、trigonometry 和 statistics。

四、重点公式速查表

Quick Reference
主题 核心形式 / 公式 学习提醒
polynomial f(x)=a_nx^n+...+a_1x+a_0；zeros、factors、end behavior 次数 degree 决定整体形状；leading coefficient 影响左右端行为。
exponential y=a·b^x，b>0, b≠1；growth / decay b>1 是增长，0<b<1 是衰减；注意 initial value。
logarithm log_b(x)=y ⇔ b^y=x；log_b(MN)=log_bM+log_bN logarithm 是 exponential 的 inverse；真数 argument 必须大于 0。
sine/cosine y=A sin(B(x-C))+D；period =2π/|B| A 是 amplitude，C 是 phase shift，D 是 midline。
unit circle 点坐标 (cos θ, sin θ)；sin²θ+cos²θ=1 先记特殊角 0, π/6, π/4, π/3, π/2，再用象限判断正负号。
normal distribution z=(x-μ)/σ；68-95-99.7 Rule z-score 表示离平均数多少个 standard deviations；图像面积表示概率。

五、十年级学生学习建议

Study Habits

每周学习安排

建议每周至少安排 4 次短时段学习：2 次跟进作业，1 次整理错题，1 次预习或补弱项。每次 35 到 50 分钟即可，重点是稳定节奏。遇到测试周，把公式表、错题和课堂 quiz 放在一起复盘。

如何预习

预习时不要追求全会。先看本节 vocabulary，例如 asymptote、period、inverse、z-score；再试着画 parent function 或 unit circle。上课时把老师强调的限制条件、图像特征和解题步骤补进去。

如何做错题本

错题本不用抄整页。每题记录四件事：题型、错因、正确关键步骤、下次提醒。比如“log equation 忘了检查 extraneous solution”比“我不会 log”更有价值。

如何为 Precalculus / AP Statistics 做准备

想走 Precalculus 或 AP Calculus，重点练函数图像、transformations、trigonometric functions 和 equation solving。想走 AP Statistics，重点练 distribution、sampling、simulation、normal model 和用一句话解释结果。
